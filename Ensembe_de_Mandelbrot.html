Programme de Renaud HENRY basé sur les sources de macsou01

Pour zoomer sur l'image, deux façons de faire :
- Entrer les coordonnées dans le formulaire.
- Définir la zone à zoomer en la dessinant sur l'image.
Pour sauvegarder l'image :
- Faire uu clic droit sur l'image et choisir "enregistrer l'image sous"
En théorie on peut zoomer à l'infini, mais en pratique, à cause de la précision de calcul limitée, l'image pixelise avec un zoom très important.
Si, après avoir beaucoup zoomé, l'image devient sombre, augmentez la précision.

Le temps de calcul peut être assez long.
Si vous avez le message suivant (sous Firefox) :
Un script sur cette page est peut-être occupé ou ne répond plus. Vous pouvez arrêter le script maintenant, l'ouvrir dans le débogueur ou le laisser continuer.
Cliquez sur Continuer pour avoir l'image en entier, sinon cliquez sur Arrêter le script.
